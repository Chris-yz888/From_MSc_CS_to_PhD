# Tools & Links

申博过程中使用的工具、指南和参考项目。链接会持续整理，避免只收藏、不消化。

## 类似 GitHub 项目

- [womanreading7/PhD-Application-Guide](https://github.com/womanreading7/PhD-Application-Guide)
  - 包含申请时间线、选校、SOP、推荐信、套磁与面试经验。
- [Dieselmarble/PhD-Application](https://github.com/Dieselmarble/PhD-Application)
  - 按 CV、SOP 和 Research Statement 等申请材料分类。
- [chinasatokolo/csGraduateApps](https://github.com/chinasatokolo/csGraduateApps)
  - 面向计算机专业研究生申请的资源合集。
- [gwisk/gradguide](https://github.com/gwisk/gradguide)
  - STEM 博士申请经验与建议。
- [CS Graduate School Application Guide](https://csguides.github.io/grad-school-application-guide/)
  - 涵盖科研准备、推荐信、文书和面试的完整指南。

## 信息检索

- [Google Scholar](https://scholar.google.com/)：搜索论文、作者及引用信息。
- [DBLP](https://dblp.org/)：查询计算机领域作者和论文记录。
- [CSRankings](https://csrankings.org/)：按研究方向查看计算机院系和教师。
- [Connected Papers](https://www.connectedpapers.com/)：探索论文之间的关联。
- [Semantic Scholar](https://www.semanticscholar.org/)：检索和追踪学术文献。

## 写作与管理

- [Overleaf](https://www.overleaf.com/)：使用 LaTeX 编写 CV 和研究材料。
- [Zotero](https://www.zotero.org/)：管理论文、引用和阅读笔记。
- [LanguageTool](https://languagetool.org/)：检查英文语法与表达。

## 使用原则

- 参考结构，不照搬他人的申请材料。
- 阅读经验帖时注意申请年份、专业和个人背景差异。
- 公开内容前移除邮箱、电话、申请编号和未公开通信。
